# 🌍 Current Projects

## 🏛️ Civilization Classics Collection

The Civilization Classics Collection is a preservation project dedicated to making the classic Civilization games easy to install, play and mod on modern versions of Windows.

Features include:

- 🖥️ Custom launcher
- 📦 Integrated installer
- 🎮 Mod Manager
- 📝 Metadata editor
- 🖼️ Preview images
- 🔄 Auto-updater
- 👥 Foundations for future multiplayer features
- 🛠️ Ongoing compatibility improvements
